# Adam's Shell (ASH)

Simple shell wrote by following Stephen Brennan's tutorial [here](https://brennan.io/2015/01/16/write-a-shell-in-c/).

# Installation
1. Clone the project
2. Navigate to the cloned folder
```
   cd ash
   ```
3.Compile the project
  ```
   gcc -Wall -Wextra -o ash main.c
   ```

# Builtin commands
1. Change directory
   ```
   cd
   ```
2. help
   ```
   help
   ```
3. exit
   ```
   exit
   ```
